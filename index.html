about FallHarbor

FallHarbor is a sovereign estate directory. The source of truth is manifest.json in this repo · this page reads it and renders the directory you see.

Every tool declares its category, capabilities, license, live URL, source repo, and — critically — forks_from, so provenance is machine-readable. A tool with many downstream forks is load-bearing substrate for the estate.

Any operator can publish a compatible manifest. Fork this repo, edit manifest.json, publish via Pages, and other FallHarbor deployments can add you to peer_estates.

The manifest is optionally signed with FallSignature (Ed25519 · Web Crypto native). The signature slot is provenance.signature.
